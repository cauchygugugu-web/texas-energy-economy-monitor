# Texas Energy & Economy Monitor

A reproducible Python project for retrieving and analyzing
Texas electricity market and economic data from public APIs.

## Current progress

- Connected to the EIA API
- Retrieved monthly Texas residential electricity prices
- Cleaned API responses into a pandas DataFrame
- Added basic data validation
- Created a time-series visualization
- Generalized the electricity price fetcher by state,
  sector, and date range

## Data source

Electricity data are retrieved from the U.S. Energy
Information Administration API.

## Project status

This project is under active development.
